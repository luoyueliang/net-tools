推荐客户端：mihomo（原 Clash.Meta）

mihomo 是高性能规则代理内核，支持 SOCKS5/HTTP/VLESS+Reality 等多种协议。配合图形化管理面板使用，是目前最强大的代理客户端方案。

🍎

服务端管理工具：mihomo-ctl

mihomo-ctl 是我们自研的 Mihomo 多平台管理 CLI 工具，单文件 Node.js 脚本，支持启停/热重载/模式切换/流量监控/节点选择/GeoIP更新/无缝升级等。

🛠️

mihomo-ctl — CLI 管理工具

支持 macOS / Linux / FreeBSD / Alpine / OpenWrt / Windows，通过 install.js 一键安装。
