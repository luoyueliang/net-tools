📖 客户端使用说明

1. 下载并安装客户端

选择适合你操作系统的客户端，从上方链接下载并安装。

2. 生成并导入配置

在上方"配置生成器"中填入你的账户信息（或点击"从服务器获取"自动填入），生成 config.yaml 后复制或下载。

  • ClashX.Meta / Clash Verge Rev：打开客户端 → 配置 → 导入本地文件 → 选择下载的 config.yaml
  • mihomo 命令行：将 config.yaml 放入 ~/.config/mihomo/ 目录，执行 mihomo -d ~/.config/mihomo
  • Stash：配置文件 → 从 URL/本地导入 → 选择配置文件

3. 开始使用

客户端启动后，本机 HTTP 代理默认监听 127.0.0.1:7890，SOCKS5 监听 127.0.0.1:7891。在浏览器或系统中设置代理即可。多数客户端支持"系统代理"一键打开。
