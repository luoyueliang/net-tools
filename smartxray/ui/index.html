📧 SMTP 邮件配置 （自助申请验证码发送）
支持标准 SMTP 和 AWS SES SMTP。SES 用户填写 IAM SMTP 凭证（SES → SMTP Settings → Create SMTP credentials）。
防火墙与端口管理
启用后添加/删除用户时会自动同步对应防火墙端口。自助申请功能要求此项非"未启用"。
需要 VPS 上已安装 aws-cli 并配置好凭证
内网提供服务时，新建用户端口会自动在 OpenWrt 上添加 DNAT 跳转规则供外网访问
🔢 端口区间配置 （管理员创建的固定用户 与 自助注册用户 使用独立端口区间）
固定用户和自助用户的端口区间完全隔离，防止自助临时账户占用永久用户端口。修改后仅影响新创建的用户。
